# How to push the updated code on github

## Step 1 (Optional)
`cd Technoholic-Quiz`

 It is optional if you are not on this directory already

## Step 2
`git add .` or  `git add <FileName>`

To add the updated files

## Step 3
`git commit -m "Your message"`

To finalize the changes with an appropriate message

## Step 4
`git push origin main`

To send the code from your local pc to github
